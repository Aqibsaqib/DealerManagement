# DealerManagement
Simple web application for managing a used car inventory

# Languages/Frameworks
-ASP.net Core MVC\
-Bootstrap\
-Javascript

# Features
-User Authentication\
-Add New Inventory\
-Edit Inventory\
-Delete Inventory\
-Sell Inventory\
-View Sold Inventory/Profit
